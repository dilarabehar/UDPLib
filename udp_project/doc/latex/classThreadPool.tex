\doxysubsection{Detailed Description}
A thread pool for processing messages with worker threads. 

This class represents a thread pool that can process messages using multiple worker threads. It allows messages of type \mbox{\hyperlink{classMsgQueueItem}{Msg\+Queue\+Item}} to be pushed to the pool for parallel processing.
