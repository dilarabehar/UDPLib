\doxysubsection{Detailed Description}
\mbox{\hyperlink{classProcessModule}{Process\+Module}} class for processing messages from a queue. 

The \mbox{\hyperlink{classProcessModule}{Process\+Module}} class provides functionality for popping messages from a message queue and processing them in a thread-\/safe manner. It is used to retrieve messages from a queue and perform further operations on them.
