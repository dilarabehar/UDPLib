\doxysubsection{Detailed Description}
A class to represent a message item. 

This class is used to represent a message item, which includes information such as destination IP and port, source IP and port, the message buffer, and the length of the message. Instances of this class are typically used to encapsulate messages before they are sent to a server or processed by a message queue.
